SIRIUS Module

A module that provides high-level Sirius functions to be used in any language supported by EASE.

Method Overview

Method Description
configureToModeling() Convert the given project to a ModelingProject.
createSession() Create a Sirius Session for the provided model file.
getModelRoot() Get the root of the EMF model displayed in the given representation.
getRepresentation() Get the representation at the given index created in the provided Sirius session.
getViewpointsFor() Get the Viewpoint that can be applied to the provided model file.
invokeAdd() Invoke an EMF 'Add' operation to add an element to the given feature of the given container.
invokeSet() Invoke an EMF 'Set' operation to set a value of an EMF model element.
loadExistingSession() Load the Sirius Session already opened in the given project.

Methods

configureToModeling

org.eclipse.sirius.business.api.modelingproject.ModelingProject configureToModeling(org.eclipse.core.resources.IProject project)

Convert the given project to a ModelingProject. This should be called somewhere in a WorkspaceModifyOperation.

Parameter Type Description
project org.eclipse.core.resources.IProject

Returns:org.eclipse.sirius.business.api.modelingproject.ModelingProject ... the project as a ModelingProject.

createSession

org.eclipse.sirius.business.api.session.Session createSession(org.eclipse.core.resources.IFile modelFile)

Create a Sirius Session for the provided model file. It uses the first available Viewpoint that can be applied to the model file and applies it. If initialization is set to true in the representation definition, then Sirius will create automatically.

Parameter Type Description
modelFile org.eclipse.core.resources.IFile the file containing the EMF model

Returns:org.eclipse.sirius.business.api.session.Session ... the created Sirius session

getModelRoot

org.eclipse.emf.ecore.EObject getModelRoot(org.eclipse.sirius.business.api.session.Session session, org.eclipse.sirius.viewpoint.DRepresentation representation, java.lang.String modelFileExtn)

Get the root of the EMF model displayed in the given representation.

Parameter Type Description
session org.eclipse.sirius.business.api.session.Session
representation org.eclipse.sirius.viewpoint.DRepresentation
modelFileExtn java.lang.String the extension of the file containing the EMF model

Returns:org.eclipse.emf.ecore.EObject ... the loaded object, or null if not found

getRepresentation

org.eclipse.sirius.viewpoint.DRepresentation getRepresentation(org.eclipse.sirius.business.api.session.Session siriusSession, [org.eclipse.sirius.viewpoint.description.Viewpoint index])

Get the representation at the given index created in the provided Sirius session.

Parameter Type Description
siriusSession org.eclipse.sirius.business.api.session.Session
index org.eclipse.sirius.viewpoint.description.Viewpoint the index of the representation in the list of all representations in the given session
Optional: defaults to <0>.

Returns:org.eclipse.sirius.viewpoint.DRepresentation ... the representation at the given index, or null if the index is too high

getViewpointsFor

java.util.Set getViewpointsFor(org.eclipse.core.resources.IFile modelFile)

Get the Viewpoint that can be applied to the provided model file. It relies on the file extension to compute the available viewpoints.

Parameter Type Description
modelFile org.eclipse.core.resources.IFile

Returns:java.util.Set ... a collection of viewpoints

invokeAdd

org.eclipse.emf.ecore.EObject invokeAdd(org.eclipse.sirius.business.api.session.Session session, org.eclipse.emf.ecore.EObject container, java.lang.String feature, java.lang.String instanceName)

Invoke an EMF 'Add' operation to add an element to the given feature of the given container.

Parameter Type Description
session org.eclipse.sirius.business.api.session.Session
container org.eclipse.emf.ecore.EObject the container that will contain the created model element
feature java.lang.String the name of the containment feature to which the new element will be added
instanceName java.lang.String the class name of the object to be instantiated

Returns:org.eclipse.emf.ecore.EObject ... the created EObject

invokeSet

void invokeSet(org.eclipse.sirius.business.api.session.Session session, org.eclipse.emf.ecore.EObject container, java.lang.String feature, java.lang.Object value)

Invoke an EMF 'Set' operation to set a value of an EMF model element.

Parameter Type Description
session org.eclipse.sirius.business.api.session.Session
container org.eclipse.emf.ecore.EObject the element that will be modified
feature java.lang.String the name of the feature that will be modified
value java.lang.Object the value to be set

loadExistingSession

org.eclipse.sirius.business.api.session.Session loadExistingSession(org.eclipse.core.resources.IProject project)

Load the Sirius Session already opened in the given project.

Parameter Type Description
project org.eclipse.core.resources.IProject

Returns:org.eclipse.sirius.business.api.session.Session ... the session, or null if none
